문제 설명

최고차항의 계수가 ½인 삼차함수 f(x)와 실수 t에 대하여 방정식 f'(x)가 닫힌구간 [t, t+2]에서 갖는 실근의 개수를 g(t)라 할 때, 함수 g(t)는 다음 조건을 만족시킨다.

(가) 모든 실수에 a에 대하여 lim{t→a+} g(t) + lim{t→a-} g(t) = 2이다.

(나) g(f(1)) = g(f(4)) = 2, g(f(0)) = 1

f(5)의 값을 구하시오. [4점]
